1.Write a blog on Difference between HTTP1.1 vs HTTP2
ans: Multiplexing: HTTP/1.1 loads resources one after the other, so if one resource cannot be loaded, it clicks all the other resources behind it.
  In contrast, HTTP/2 is able to use a single TCP connection to send multiple streams of data at once so that no one resource blocks any other resource.
  
2.Write a blog about objects and its internal representation in Javascript
ans: Loosely speaking, objects in JavaScript may be defined as an unordered collection of related data, of primitive or reference types, in the form of “key: value” pairs.
  These keys can be variables or functions and are called properties and methods, respectively, in the context of an object.
